## .exe 파일 생성하기
- pyinstaller 설치  
```
pip install pyinstaller
```  
- onefile로 .exe 생성  
```
pyinstaller -F main.py
```  

## Git Bash 도움말
- github 저장소 로컬에 복제하기  
```
git clone https://github.com/yesung05/24-OSS-Teamwork
```  
- 로컬 깃 저장소 github 파일로 최신화  
```
git pull
```  
- 로컬 깃 저장소의 내용을 github로 업로드  
```
git push
```
